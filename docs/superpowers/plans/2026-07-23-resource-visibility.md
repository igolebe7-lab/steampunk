# План реализации видимости ресурсов

**Цель:** сделать запасы главного склада и состав выбранного здания однозначно
читаемыми без изменения симуляции.

## 1. Защитные тесты

- Расширить интеграционный тест HUD проверкой дерева, железа, угля и воды.
- Расширить тест инспектора проверкой состава склада и резервов.
- Добавить новые ключи в проверку локализации.

## 2. Верхняя панель

- Заменить единственную метку древесины четырьмя компактными метками ресурсов.
- Передать их в `HUDController`.
- Обновлять значения в порядке каталога из главного склада.

## 3. Инспектор

- Сформировать детерминированный список релевантных ресурсов здания.
- Для склада показывать весь каталог, для остальных зданий — контекстный набор.
- Вывести фактический запас и ненулевые входящие/исходящие резервы.

## 4. Локализация и проверка

- Добавить русские и английские подписи.
- Импортировать переводы и выполнить целевые тесты.
- Выполнить полную проверку проекта и краткий визуальный smoke-тест.
- Зафиксировать `PT-R01` как незачётный из-за дефекта наблюдаемости.
